# 🐍 贪吃蛇（独立窗口版）

一个用Python + Pygame做的经典贪吃蛇，支持  
- ✅ 方向键控制  
- ✅ 不需要终端，可以在独立窗口运行  
- ✅ 排行榜自动保存（本地 top5）  
- ✅ 速度随分数越来越快
